人生十八局

由华华分享

人生十八局

收藏本棋书

记录了吴清源先生一生经典的十八局，并配有棋谱和讲解。让读者再次领略围棋大师的风范。
通过对本书的记述和编辑，让人感受最深的一点就是，这几十年间，布局和定式发生了极其显著的变化。

即使像吴清源老师这样的天皇巨星，也会对过去的对局产生出今是而昨非的反思――如果是现在的话，我将这样下――遑论我们这些平庸的后学。布局和定式不仅可以变化得和过去完全不一样，而且，对其评价有时也会和过去完全相反。

按照这样的观点，今后，围棋也许仍然会不断地进步发展，就像当初，正是由于出现了吴清源老师这样天才加勤奋的人才，围棋才得到了进一步的发展。随着时间的演变，围棋一定会不断地进步发展。

现在，本书收录了吴清源老师围棋生涯中的一部分对局，由吴老师自己亲口进行解说。从这个意义上讲，本书走在了时代的前面，它不仅仅是针对当代人的，同时，也是留给后世的一笔宝贵财富。

围棋是中和――吴老师总是将这句话挂在嘴边。当你凝视着整个棋盘时，吴老师行棋时那行云流水般自然的调子，是真正的围棋的范本。围棋到底应该是怎样下的，行棋的方向到底应该下在哪里才是正确的――仅仅将本书的棋谱摆一遍，你就会明白。

来日本以后，我和木谷先生下了许多盘棋，我总是输棋。在没有贴目的升段赛中，我执黑本来很少输棋，但和木谷先生对局时却总是不尽人意。

木谷实
1909年出生神户。1921年成为铃木为次郎的入门弟子，1924年入段。1926年秋三段。"新进选拔赛"连胜十人，被称为"怪童丸"。1933年与吴青源共同开始新布局的实践，由此引发了昭和棋界的大变革。1938年成为秀哉名人引退棋的对战者，执黑五目胜。1962年门下突破百段。1965年获紫绶褒章。1975年12月去世，终年66岁。叙勋从四位勋二等瑞宝章。木谷一门的弟子成为支撑日本现代棋界的栋梁。

小野田先生是日本棋院的元老。他很年轻的时候就去世了，去世前棋力犹如鬼神般强大可怕，被称作“鬼田强太郎”。

1896年出生于东京。1914年经方圆社认可为初段。1920年五段。1923年加入中央棋院，与小岸壮二一起从事"棋院新报"的创办活动。1939年七段。1942年秋季、1943年春季两期"大手合"升段赛连续取得全胜冠军。别名"鬼田强太郎"，据说晚年的力量已近鬼神。作为登山家也非常有名，几乎踏破日本只要叫得出名字大山峰。1944年去世，终年48岁。同年追赠八段。

1907年2月27日出生于大阪。1920年成为濑越宪作名誉九段的入门弟子。1929年入段。1954年九段。第2、5、6期本因坊。称号"昭宇"。1950年创立关西棋院。十段位二期，王座位三期。1977年秋季，被授予勋三等旭日中绶章。关西棋院理事长。1994年7月24日去世，终年87岁。
岩本·吴十番棋
熏和 VS 吴清源
这是和当时的本因坊、前辈岩本熏八段十番棋的第一局。
我还是孩子的时候，在北京就见到过岩本先生，让三子局我赢了，让二子局我 输了二目，这成为我为日本棋界所知的契机。

本因坊熏和
1902年2月5日出生于岛根县。原名岩本熏。1913年成为广濑平治郎八段的入门弟子。1917年入段。1946年成为第三期本因坊。2947年第四期本因坊战击退木谷八段的挑战连霸。1961年赴美国从事围棋普及的活动，从此将全部精力倾注于海外围棋的普及。1967年获紫绶褒章。1972年获勋三等瑞宝章。1999年11月29日去世，终年97岁。
